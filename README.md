A Book Store App which created React Native CLI

# <sub>**Topics**</sub>
- React Native CLI
- Redux Tool Kit
- React Navigation
- Axios
- Responsive Design

# <sub>**Getting Started**</sub>
First, install node modules 

`npm install`

Then, start Metro Bundler

`npx react-native start`

Finally, run the app

`npx react-native run-android`
